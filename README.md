# Nikefa — Medical Supply Marketplace

تطبيق متجر إلكتروني للمستلزمات الطبية، مبني بـ Flutter.

## التقنيات المستخدمة
- **Flutter** — تطبيق موبايل + ويب
- **Supabase** — قاعدة بيانات ومصادقة
- **Riverpod** — إدارة الحالة
- **GoRouter** — التنقل بين الصفحات

## المميزات
- متصفح منتجات طبية
- سلة مشتريات
- بحث وتصفية
- حسابات مستخدمين
- دعم عربي كامل
